# 🌋 Global Volcano Explorer

**Global Volcano Explorer** is a futuristic and immersive landing page that showcases some of the most iconic volcanoes around the world. It visually educates users about different types of volcanoes — Stratovolcanoes, Shield Volcanoes, and Caldera Volcanoes — through beautifully styled cards, high-quality images, and elegant typography. Designed with a sleek aesthetic and 3D transitions, this page makes Earth science look stunning.

---

## 🌟 Features

- ✅ Categorized volcanoes: Stratovolcanoes, Shield Volcanoes, Caldera Volcanoes
- ✅ 3D hover effects on volcano cards
- ✅ Transparent (glassmorphism) design using `backdrop-filter`
- ✅ Elegant Google Fonts: `Playfair Display` for headings, `Orbitron` for modern body text
- ✅ Dark-themed gradient background for futuristic appeal
- ✅ Clean responsive layout using Flexbox
